# Bmob-vue
一个使用bmob+vue-cli+npm的简单博客，带有文章发布点赞评论查看等功能

技术栈Vue-Cli+Bmob

### 开始
``` javascrip
npm install
```

### 启动
``` javascript
npm run build
```

### 打包
``` javascrip
npm run build
```
