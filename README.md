一个使用`bmob`+`vue-cli`+`npm`的简单博客，带有文章发布点赞评论查看等功能

技术栈`Vue-Cli`+`Bmob`

## 使用规范

### 引入Bmob
``` javascript
import Bmob from "hydrogen-js-sdk";
```

### 修改key

``` javascript
Bmob.initialize(
    "Application ID",
    "REST API Key"
);
```

### 开始
``` javascrip
npm install
```

### 启动
``` javascript
npm run build
```

### 打包
``` javascrip
npm run build
```
